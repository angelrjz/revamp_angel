# INSTRUCCIONES — 5 MINUTOS

## Qué tienes
PWA (Progressive Web App) que vive en tu iPhone. No necesita backend ni backend ni nada.

## Qué necesitas
- Cuenta de GitHub (gratis, 5 min para crear)
- Cuenta de Vercel (gratis, conecta con GitHub)

## Los 5 pasos

### Paso 1: GitHub (cuenta + repo)
1. Ve a [github.com/signup](https://github.com/signup)
2. Crea tu cuenta (email, contraseña, nombre de usuario)
3. Ve a [github.com/new](https://github.com/new)
4. Nombre del repo: `centro-operaciones`
5. Public (checkbox)
6. Click "Create repository"
7. **Copia el link que te da** (parecido a: `https://github.com/tu-usuario/centro-operaciones.git`)

### Paso 2: Descompactar + Subir archivos
1. Descompacta `centro-pwa.zip` en tu computadora
2. Abre GitHub Desktop (o línea de comandos)
3. Clone el repo que acabas de crear
4. Copia los archivos del ZIP a la carpeta clonada
5. En GitHub Desktop:
   - Tab "Changes"
   - Write: `Initial commit`
   - Click "Commit to main"
   - Click "Push origin"

### Paso 3: Vercel (1 click)
1. Ve a [vercel.com](https://vercel.com)
2. Click "Sign up" → elige "Continue with GitHub"
3. Autoriza
4. Click "Add New" → "Project"
5. Busca `centro-operaciones`
6. Click "Import"
7. Espera 30 segundos
8. **Copia el link que aparece** (ej: `centro-operaciones.vercel.app`)

### Paso 4: iPhone
1. Abre Safari en tu iPhone
2. Pega el link de Vercel
3. Espera a que cargue
4. Botón "Compartir" (abajo) → "Añadir a pantalla de inicio"
5. Nombre: "Centro Ops" (o lo que quieras)
6. Click "Añadir"

### Paso 5: Listo
¡Ya está! Tienes la app en tu home con icono. Todos tus datos se guardan en el iPhone (localStorage).

---

## ¿Datos? Todo en tu iPhone

Los datos NO se suben a ningún servidor. Todo queda local en el teléfono. Seguro, rápido, offline.
